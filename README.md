### Agents

| Agent | Description |
|---|---|
| [pump-quant](https://github.com/pumpdotstudio/pump-quant) | Quant agent starter — deterministic heuristic analysis of Solana memecoins |
| [intern](https://github.com/pumpdotstudio/intern) | Automated screenshotter + quant ranker, 10x daily on GitHub Actions |
| [allah](https://github.com/pumpdotstudio/allah) | Self-replicating agent fleet — spawns 5 new agents daily, ranks across entire fleet |
| [agent-zero](https://github.com/pumpdotstudio/agent-zero) | BiP Hackathon agent |
| [trainer](https://github.com/pumpdotstudio/trainer) | Training software for agent models |
